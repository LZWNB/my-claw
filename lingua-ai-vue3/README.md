# LinguaAI - AI英语学习平台

一个面向东亚用户的AI英语学习网站，支持多语言和暗黑模式。

## 功能特性

- 🌍 多语言切换（简体中文、繁体中文、日语、韩语）
- 🌓 白天/黑夜模式切换
- 🏠 主页展示
- 👤 用户登录/注册
- 🤖 AI对话练习
- 📚 个性化课程定制
- 📖 词库 + 单词搜索
- 🎯 背单词功能
- 🔥 个人打卡 + 好友打卡
- 📊 可视化学习进度

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- i18next（国际化）
- Zustand（状态管理）
- React Router（路由）
- Recharts（图表）

## 开发命令

```bash
npm install
npm run dev
npm run build
```

## 项目结构

```
src/
├── components/     # 通用组件
├── pages/          # 页面组件
├── hooks/          # 自定义Hooks
├── stores/         # 状态管理
├── i18n/           # 国际化配置
├── utils/          # 工具函数
├── types/          # TypeScript类型
└── assets/         # 静态资源
```
